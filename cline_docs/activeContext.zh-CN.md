# 活动上下文：

**用途：** 本文件提供当前工作重点、即时后续步骤和项目活动决策的简明概述。旨在作为一个经常参考的高层次摘要，以保持项目动力和团队一致性。

**使用指南：**
- **当前工作重点：** 列出当前正在积极推进的 2-3 个*最关键*的任务。保持描述简洁且面向行动。
- **后续步骤：** 列出推进项目所需的即时后续步骤。为清晰起见，对这些步骤进行优先级排序和排序。
- **活动决策和考虑因素：** 记录当前正在考虑或积极讨论的关键决策。捕捉决策的本质和任何未决问题。
- **不要包括：** 详细的任务分解、历史变更、长期计划（这些属于其他记忆库文件，如 `progress.md` 或专用文档）。
- **保持简洁：** 保持此文件简洁并专注于项目的*当前*状态。定期审查并修剪过时的信息。
